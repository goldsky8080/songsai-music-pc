# 개발 / 운영 가이드

## 로컬 개발

### 프론트

```powershell
cd D:\songsai-music-pc
npm run dev
```

### 백엔드

```powershell
cd D:\wrapper\suno-api
npx prisma generate
npx next dev --port 3100
```

## 로컬 주소

- 프론트: `http://localhost:3000`
- API: `http://localhost:3100`

## 프록시 원칙

- 브라우저는 직접 `3100`을 치지 않는다
- 프론트에서 `/api/proxy/...`를 통해 백엔드로 전달한다
- 이유:
  - 로컬 쿠키/세션 안정화
  - CORS 문제 완화

## 환경값

### 프론트 `.env.local`

```env
NEXT_PUBLIC_SONGSAI_API_URL=http://localhost:3100
```

### 백엔드 `.env`

- `APP_URL=http://localhost:3100`
- `FRONTEND_URL=http://localhost:3000`
- Suno 연동용 cookie 및 키

## 배포 기본 원칙

1. 로컬에서 기능 확인
2. Git 반영
