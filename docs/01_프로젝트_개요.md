# 프로젝트 개요

## 목표

`songsai-music-pc`는 SongsAI의 PC 웹 프론트입니다.

- 프론트: `D:\songsai-music-pc`
- 백엔드: `D:\wrapper\suno-api`
- 참고 프로젝트: `D:\music`

핵심 방향은 `songsai-api`를 서비스 원본 백엔드로 키우고, 프론트는 얇게 유지하는 것입니다.

## 저장소 역할

### `D:\songsai-music-pc`

- PC 전용 프론트엔드
- 홈, 생성 화면, 내 자산, 가격, 지원 화면 담당
- 브라우저에서는 가능한 한 `/api/proxy/...`만 호출

### `D:\wrapper\suno-api`

- 인증, 세션, Google 로그인
- 음악 생성 요청, 상태 갱신, 다운로드
- `GenerationJob` 큐 처리
- 이후 비디오 렌더와 라이선스 발급까지 흡수

### `D:\music`

- 과거 메인 프로젝트
- 음악/비디오/자산 구조 참고용
- 직접 원본으로 삼지 않고 필요한 도메인 로직만 이관

## 현재 서비스 원칙

- 홈은 완료된 곡만 보여준다.
- `My Assets`에서는 진행 중인 곡도 보여준다.
- 음악 1회 생성 시 보통 2곡이 오므로, 최종 UI는 2트랙 묶음 구조를 지향한다.
- 비디오는 provider가 주는 URL을 기다리지 않고 우리 서버에서 생성한다.
