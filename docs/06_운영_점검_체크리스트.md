# 운영 점검 체크리스트

## 재부팅 후 점검 순서

### 1. SSH 접속

```bash
ssh -p 2233 songs@125.189.206.134
```

### 2. 내부 / 공인 IP 확인

```bash
hostname -I
curl ifconfig.me
```

### 3. pm2 상태 확인

```bash
pm2 status
```

기대:

- `songsai-api` online
- `songsai-music-pc` online

### 4. nginx 상태 확인

```bash
sudo systemctl status nginx
```

### 5. 로컬 포트 확인

```bash
curl -I http://127.0.0.1:3100
curl -I http://127.0.0.1:3000
```

### 6. 외부 도메인 확인

```bash
curl -I https://api.songsai.org
curl -I https://pc.songsai.org
```

## 복구 기본 명령

```bash
pm2 resurrect
sudo systemctl restart nginx
pm2 save
```

## 자동 복구 설정

`pm2 startup` 과 `pm2 save`가 설정되어 있어야 한다.
